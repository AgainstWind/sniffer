#swordSniffer
****   

##Introduction

**swordSniffer** is a graphical network packet sniffer.

##Develop environment

- Gtk+3.0
- Glade3.0
- libpcap

##Build

- git clone https://github.com/onestraw/sniffer
- cd sniffer
- make
- sudo ./sniffer
